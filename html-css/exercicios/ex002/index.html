Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito basta colocar o texto no meio das tags porém seu texto não vai quebrar as linhas sozinho, então é necessário usar o br.

Vamos testar como a tag br funciona, vamos apoveitar e usar dois símbolos < e >
Não é recomendado usar o br para dar espaçamento

Ou usar a tag p para comecar um novo parágrafo na linha de baixo. Além do espaçamento que a tag p seta das demais linhas.
